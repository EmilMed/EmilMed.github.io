My Projects

AI Assisted Cybersecurity Threat Detection System

In this project, I developed a machine learning model to detect DDoS attacks using the CIC-IDS-2017 dataset. I carefully preprocessed the data, identified key features like Total Backward/Forward Packets and Fwd/Bwd ratio, and used SMOTE to address class imbalance. I trained and validated a Random Forest classifier through cross-validation, ensuring it could perform robust, real-time DDoS detection. The project utilized Python, Scikit-Learn for modeling, and Seaborn for visualization, demonstrating my expertise in machine learning, data preprocessing, and cybersecurity.

CONNECT (Advanced Meeting Reminder System)

Built "CONNECT," a full-stack web application using Django and PostgreSQL, designed to enhance user connections through an advanced meeting reminder system. Users can customise notification settings, including frequency and availability, ensuring meeting suggestions align with their schedules. Integrated Celery for asynchronous task management, enabling automated, timely reminders based on user preferences. The intuitive interface, built with HTML/CSS, JavaScript, and FullCalendarJS, allows users to manage contacts, schedule meetings, and view events on a dynamic calendar.

Fitness Advisor

The web-based gym assistant I developed is designed to incorporate user input, such as Body Mass Index (BMI), to offer a personalized and versatile fitness program. Utilising Flask, I managed the back-end of the website, ensuring a secure user authentication system. For the front end, I used JavaScript, HTML, and CSS to structure the content effectively while enhancing the platform's responsiveness and aesthetics.
